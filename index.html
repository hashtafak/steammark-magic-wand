Formatting Guide

Markdown to Steam BBCode Conversion

This tool converts standard Markdown formatting to Steam-compatible BBCode. Here's what gets converted:

Headers

# Header 1 → [h1]Header 1[/h1]

## Header 2 → [h2]Header 2[/h2]

### Header 3 → [h3]Header 3[/h3]

Text Formatting

*italic* → [i]italic[/i]

**bold** → [b]bold[/b]

~~strikethrough~~ → [strike]strikethrough[/strike]

Lists

- Item 1 → [list][*]Item 1[/list]

1. Item 1 → [olist][*]Item 1[/olist]

Links & Images

[text](url) → [url=url]text[/url]

![alt](src) → [img]src[/img]
